# BrainfuckRobot
Send me some brainfuck code and I will send you the output.

/help: Display this message.
/brainfuck: Get basic info on how brainfuck works.
/minify: Minify brainfuck code that contains comments.
/run: Execute your brainfuck code. E.g.: /run ++++.--.
/debug: Debug your brainfuck code. Returns the values of the pointers after executing the code. E.g.: /debug ++++.--.
